# Changelog

## v1.0.0 — 2026-04-16

### Added
- Initial release
- FastQC, Kraken2, SPAdes (--metaviral) steps
- VirSorter2 viral sequence detection
- CheckV completeness & quality assessment
- Circular/complete genome filter (DTR/ITR + completeness threshold)
- BACPHLIP lytic/lysogenic lifestyle prediction
- Phrokka annotation for *E. coli* phages
- Prokka annotation for *Listeria*, *Salmonella*, *Enterococcus*, *Staphylococcus* phages
- vContact2 viral taxonomy clustering
- vHULK host prediction
- Optional BLAST nucleotide search
- MultiQC aggregated QC report
- Conda, Docker, Singularity, SLURM profiles
- Parameter validation and `--help` flag
